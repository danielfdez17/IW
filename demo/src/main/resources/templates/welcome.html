¡Adivina el Número!

Objetivo del Juego

El objetivo es adivinar el número secreto seleccionado por el sistema entre las opciones disponibles.

Cómo Jugar

Elige un número entre las opciones mostradas en pantalla.
¡El sistema te dirá si el número que seleccionaste es mayor, menor, o si has acertado!
¡Sigue intentando hasta encontrar el número correcto!

Opciones Seleccionables

El juego mostrará un rango de números del 0 al 10.
